# Erudite
Программа «Эрудит» представляет собой компьютерную реализацию классической настольной игры, в которой участники составляют слова из буквенных фишек на специальном игровом поле и на одном компьютере. 
Для запуска игры используется исполняемый файл Erudite.exe.
После запуска программы пользователь попадает в главное меню. При выборе новой игры открывается окно настройки, где определяется количество участников (от 2 до 4 человек). Основное игровое окно разделено на две функциональные зоны. Левая часть содержит игровое поле размером 15×15 клеток, визуально окрашенное в соответствии с бонусными зонами: обычные клетки бежевого цвета, синие клетки обозначают удвоение стоимости буквы, жёлтые – утроение стоимости буквы, зелёные – удвоение стоимости слова, красные – утроение стоимости слова. Центральная клетка, помеченная тёмно-красным цветом и символом звёздочки, является стартовой позицией.
Правая часть интерфейса представляет собой панель управления, содержащую информационные элементы и кнопки действий. В верхней части отображается информация о текущем игроке, его количестве очков и числе оставшихся фишек в общем мешке. Ниже расположены четыре основные кнопки управления: «Сделать ход» для подтверждения размещения фишек, «Пропустить ход» для передачи хода следующему участнику, «Поменять буквы» для замены фишек и «Завершить игру» для досрочного окончания партии.
Процесс игры начинается с хода первого игрока, который должен составить слово, обязательно проходящее через центральную клетку поля. Для этого игрок должен кликнуть левой кнопкой мыши на одну из семи фишек в своей панели, после чего выбранная фишка выделится зелёным цветом. Затем игрок кликает на целевую пустую клетку игрового поля, и фишка туда перемещается.
Если игрок допустил ошибку при размещении, он может вернуть фишку, кликнув на неё на поле – фишка возвращается в панель игрока. 
После размещения необходимого количества фишек (от одной до семи) игрок нажимает кнопку «Сделать ход». Программа выполняет проверку: для первого хода подтверждается прохождение через центральную клетку, для последующих ходов проверяется соприкосновение новых фишек с уже существующими словами, и анализируются все горизонтальные и вертикальные слова, образовавшиеся на поле, на предмет их наличия в загруженном словаре. Если проверка проходит успешно, система подсчитывает очки, учитывая базовую стоимость букв, модификаторы бонусных клеток и дополнительный бонус в 50 очков за использование всех семи фишек. Результаты хода отображаются в информационном окне, после чего игрок автоматически добирает фишки из общего мешка до семи штук, и ход переходит следующему участнику по кругу.
Игра завершается при выполнении одного из двух условий: когда у одного из игроков заканчиваются фишки и при этом общий мешок пуст, либо когда все участники подряд пропускают ход два полных круга. По окончании игры программа отображает итоговое окно с результатами всех участников, отсортированными по количеству набранных очков. Игрок с наибольшим количеством очков объявляется победителем. При равенстве очков у нескольких участников система констатирует ничью. После ознакомления с результатами пользователь может начать новую игру, вернувшись в главное меню.
